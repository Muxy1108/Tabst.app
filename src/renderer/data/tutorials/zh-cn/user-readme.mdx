# 欢迎来到 Tabst

欢迎使用 Tabst！这里是你“写谱 + 听谱 + 管理灵感”的交互式工作台。

Tabst 基于 alphaTab，支持 `.atex / .gp / .gp3 / .gp4 / .gp5 / .gpx` 等格式的渲染与播放。

## 为什么用 AlphaTex

AlphaTex 是高表达力纯文本语法：像写 Markdown 一样写轨道、节拍、指法与效果。

## 推荐工作流

1. 先用 AlphaTex 快速起稿
2. 立即预览并试听
3. 迭代节奏、效果与结构
4. 与你的笔记体系配合沉淀创作
