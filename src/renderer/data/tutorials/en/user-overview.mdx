# AlphaTex Overview

AlphaTex is AlphaTab's text-based notation format. It typically contains:

1. Metadata (optional)
2. Musical content (required)
3. Sync points (optional)

Sections are separated by a single dot (`.`).

- Song metadata: title, artist, tempo, etc.
- Song contents: tracks/staves/bars/beats/notes
- Sync points: align score with external media (advanced)
