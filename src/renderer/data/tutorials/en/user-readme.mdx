# Welcome to Tabst

Welcome to Tabst! This is your interactive workspace for writing tabs, hearing playback, and managing musical ideas.

Tabst is built on alphaTab and supports formats like `.atex / .gp / .gp3 / .gp4 / .gp5 / .gpx` for rendering and playback.

## Why AlphaTex first

AlphaTex is expressive plain text. You can describe tracks, beats, fingering, and effects like writing Markdown.

## Typical workflow

1. Draft musical ideas in AlphaTex
2. Preview and listen immediately
3. Refine rhythm, effects, and structure
4. Keep related notes in your knowledge workflow
