# Roadmap

This document outlines the planned features and improvements for Tabst.

## Current Status

Tabst is currently in active development. We're focusing on core functionality and user experience improvements.

## Planned Features

### Short-term (Next Release)

- [ ] Enhanced tutorial system
- [ ] Improved error handling and user feedback
- [ ] Performance optimizations
- [ ] Additional documentation

### Medium-term

- [ ] OMR (Optical Music Recognition) integration
  - Convert PDF/Image scores to alphaTex
  - Batch processing support
- [ ] Advanced export options
  - Custom PDF templates
  - Multiple format support
- [ ] Collaboration features
  - Cloud sync
  - Real-time collaboration

### Long-term

- [ ] Plugin system
  - Custom extensions
  - Community plugins
- [ ] Mobile companion app
- [ ] Advanced music theory tools
  - Chord progression analysis
  - Scale suggestions
  - Key detection
